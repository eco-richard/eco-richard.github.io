Projects

While at App Academy, I completed three full stack projects which consisting of cloning three popular websites with robust and user friendly frontendsn and efficient and clear backends.

DulyNoted

DulyNoted is a clone of the popular note taking app Evernote. It was creating using a backend consisting of Python, Flask, SQLAlchemy and Alembic. The frontend was created using Javascript, React, and Redux. Users are able to create accounts and take notes, then they are able to organize their notes into notebooks.

The app also has a scratch pad that can be used as a location to store just some things that may have to be written quickly. Notes also have an autosave feature so no need to worry about forgetting to save your changes. The source code of DulyNoted can be found here and a link to the live site can be found here.

RiftPay

RiftPay is a clone of the transaction tracking app Splitwise. It was created using a Python, Flask backend and a Javascript, React, Redux frontend, similar to DulyNoted. Users are able to create transactions with their friends to keep track of expenses. For example, if three friends go out to dinner, one could just pay the entire bill and RiftPay will track the transaction and remind the two friends that they owe the one friend $33.33 each.

This clone is filled with features including creating transactions with multiple users, track balances between users, add other users as friends and add comments to specific transactions. The app has a clear use of modal menus to improve the user experience. The source code of RiftPay can be found here and a link to the live site can be found here.

AirBorNot2B

AirBorNot2B is a clone of the home hosting site AirBnB with some literature inspired spots and users. It was created using an Express and Node backend with a React, Redux frontend. Users are able to host spots and leave reviews on other spots.

The source code of AirBorNot2B can be found here and a link to the live site can be found here.
